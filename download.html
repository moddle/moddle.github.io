Installation Instructions:

To install Moddle, first download the latest version from the panel to your right, then follow these steps:

  1. Place the launcher executable in an empty folder.
  2. Start the launcher. You should be able to start the application by clicking or double-clicking the executable.
  3. If the previous step does not work, you may have to start the application via command line. Open your command line of choice, navigate to the folder containing the jarfile, and run this command: javaw.exe -jar ./Moddle.jar
  4. After the launcher starts, it will probably tell you that there are updates available. You should always install any updates that are available when you run Moddle for the first time. Click 'Yes' to install the updates.

Once everything has been downloaded and installed, you are all done! Log in and create a new modpack instance to get started playing. For more information on how to use the launcher or make your own modpacks, refer to our videos here.

If the launcher will not start with Java 8, try running the jarfile with Java 7 instead.

Also remember that it is very important for you to report any bugs that you find in the launcher on our GitHub Issue Tracker. Thanks!
