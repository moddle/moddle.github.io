Moddle Version 0.2 - September 2, 2014

Well, this is it! The very first release of the all-new Moddle launcher! Now 100% cross-platform, a graphical user interface, and more features than ever before! We are so excited to bring this application to you today... there are just no words... yeah. Many months of hard work are represented here, and we hope that you enjoy the results.

As you may or may not know, this is an in-dev launcher; it is a work-in-progress. If you find bugs, please report them on the issue tracker on GitHub. We would appreciate this greatly!

And now, the feature list:
* Completely rewritten in Java
* Cross-platform compatibility
* A graphical user interface
* Modpack logos
* Modpack descriptions
* Modpack instance support
* Yggdrasil login support
* News feed
* JSON-based configuration files
* User preferences
* A console
* Settings for everything
* Auto-configuration of compatible config files
* Auto-IDs in compatible configs
* STUUUUUUUUUFFFF
* MOAR STUUUUUUUUUFFFF
~ No permissions checking yet
~ Jarmods are not finished yet (tragically)
~ No pack-building GUI yet

Minecraft versions:
* 1.6.4

All 1.6.X and newer versions should be supported, 1.6.4 is the only version currently included. Pre-1.6 versions are still in the works due to zipfile manipulation limitations.
