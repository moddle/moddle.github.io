Supporting the Project

Do you think that this is a great project and want to help support us in its development? If you do, we have a few ways in which you can help:

Writing Code

You can help us develop the Moddle launcher by coding! Moddle is written in the Java language, using the NetBeans IDE. To program the launcher, you will need to have installed the Java JDK, NetBeans IDE, and Git/GitHub. Git via GitHub is used to distribute and manage our source code. To get our source code, fork the project repository on GitHub. Once you have made changes, create a pull request and we will look at your code and pull it into the main repository.

At the time of writing, we have not yet created any sort of source code documentation or developer's guide. Most of the code should be fairly well-commented, however. If you need help, do not hesitate to contact us. Contact information is available on the 'About' page.

Building the Cache

You can also help us out by building Cache entries! The amount of time we will have to add entries to the Cache will be inversely proportional to the amount of bugs that need fixing or features that must be added. This being so, we need people to make Cache entries for the mods that they want to be in the Cache so that we can add them to the distribution system and expand the mod list. The only limitation for making entries is permissions: We can only add mods to that Cache that have public-modpack permission. If you have made one or more Cache entries that you wish to submit, send them to us via email. We will then review them and add them to the system.

Donating

If you really like this project and you want to show your support in a more physical or financial manner, please consider donating. Donations are greatly appreciated and are also very motivating. Showing your support of our project drives us to keep working, adding features, and improving the launcher.

At this time we can only take Bitcoin donations. PayPal and other monies are still being set up... sorry!
